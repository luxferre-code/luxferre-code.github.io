LuxLib

But du projet:

LuxLIB est une librairie Python permettant la mise à jour automatique depuis un serveur. Mais aussi de pouvoir mettre à jour son serveur depuis le client.

Installation:

Pour installer LuxLIB, il faudra utiliser pip.
pip install luxlib
