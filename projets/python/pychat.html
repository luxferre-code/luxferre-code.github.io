PyChat

But du projet:

PyChat est un programme python. C'est une interface du style "discord", on peut y gerer des salons, gérer les utilisateurs si on est le créateur du salon, etc... . Dans le répertoire il y a deux dossiers, client pour utiliser PyChat, et server pour démarrer le serveur de liaison. Important, il faut changer les ip sur le client

Installation:

Il faut absolument les modules suivants:

  • pysftp
  • pygame
  • PIL
  • hashlib

Utilisation:

  • Pour le client:

    • Il suffit de démarrer le fichier main.py

  • Pour le serveur:

    • Il suffit de démarrer le fichier main.py
