PyLink Discord Bot

But du projet:

Le bot discord PyLink permet de synchroniser des channels d'un serveur avec un autre serveur discord

Installation:

Il suffit de compier le dépôt github et de modifier le fichier token_bot.py

  • Il faut aussi avoir les modules suivants
    • discord.py
    • colorama

Utilisation:

Il suffit de démarrer le fichier PyLink.py et le tour est joué :)
