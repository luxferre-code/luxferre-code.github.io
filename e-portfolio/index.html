Un peu de moi...

Je me présente: Valentin Thuillier, étudiant en 1er année à l'IUT-A de Lille filliaire Informatique. J'ai aujourd'hui 18 ans, et un véritable passionné de l'informatique. Je viens du lycée Beaupré à Haubourdin. Ou j'ai suivi la spécialité NSI (Numérique et Science de l'Informatique) et Mathématique. J'ai appris depuis la 3ème plusieurs langage de programmation, en commencant par de la programmation par blocs (Scratch). Puis j'ai commencé la programmation avec Python, puis Java

En Python

Le langage Python est mon langage de prédilection. J'ai appris les bases de la programmation avec. Et il restera je pense à jamais dans mes langages voir mon langage préféré. Python est modulable à souhait, capable de gérer presque tous: de l'affichage, du web et des bases de données. Si vous souhaitez voir tous les projets que j'ai mené dans ce langage, je vois revois directement sur mes projets python.

En Java

Le java n'est pas un langage que j'affectionne réellement. Je l'ai appris pour un jeu connu Minecraft. un jeu que je joue depuis mon enfance (j'ai commencé de mémoire en 2014). J'ai appris les bases pour faire du modding. Le modding sur Minecraft se fait onligatoirement en java, étant donné que le jeu est lui-même fait en java. J'ai donc appris les "bases" sur le site Zeste de savoir. J'ai aussi créer de multiple serveur Minecraft qui ne sont plus à ce jour allumé, en configurant les plugins, les mods, le réseau et la sécurité.
